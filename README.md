# Fitolit Kuta Baginda Berau



Project ini merupakan repository data fitolit yang diperoleh dari Situs Kuta Baginda.
Data fitolit diekstraksi dari sedimen yang diperoleh dari kegiatan ekskavasi tahun 2021.
